# 2026-06-29 V1 产品与 UI 设计变更记录

## 实现过程

- 根据项目目标设计了新闻 RSS 订阅、存储与阅读网站的 V1 功能范围。
- 明确技术栈为 Spring Boot、PostgreSQL、Vue 3、TypeScript、Vite、Naive UI。
- 设计了 RSS 源管理、解析模板、文章抓取、阅读舱、仪表盘、抓取日志等核心模块。
- 针对不同 RSS 源字段不一致的问题，设计了解析模板方案，包含字段映射、时间解析、内容清洗、封面图提取和去重策略。
- 设计了深色科技感 UI 方向，并拆分了首页驾驶舱、阅读页、订阅源管理页、解析模板页、抓取日志页。
- 设计了后端包结构、核心服务、数据库核心表和 API 草案。
